# WorkoutPage-FrontEnd-
## Overview

This is a fully responsive workout tracker web application designed for fitness enthusiasts to log, track, and visualize their exercise routines. Built with vanilla HTML, CSS, and JavaScript, it emphasizes clean UI/UX with customizable themes, interactive calendars, and real-time progress updates. No external frameworks are required—it's lightweight, performant, and easy to deploy.

The app helps users plan workouts, monitor progress over time, and switch between light/dark themes for better usability. It's ideal for personal fitness tracking or as a starting point for more advanced gym management tools.

## Key Features

- **Interactive Calendar View**: Select dates to view or add workouts, with visual indicators for completed sessions and upcoming goals.
- **Workout Logging**: Input exercises, sets, reps, weights, and notes; data persists via localStorage for offline access.
- **Progress Dashboard**: Charts and stats showing trends like total volume lifted, streak counters, and weekly summaries.
- **Customizable Themes**: Toggle between light, dark, and high-contrast modes with smooth CSS transitions.
- **Responsive Design**: Adapts to mobile, tablet, and desktop screens using flexbox and media queries.
- **Export/Import Data**: Download workout history as JSON or import from previous sessions.

## Tech Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Advanced styling with CSS Grid, Flexbox, animations, and custom properties for theming.
- **JavaScript (ES6+)**: DOM manipulation, event handling, localStorage for data persistence, and dynamic chart rendering using Canvas API.

## How to Use

1. Clone the repo and open `index.html` in any modern browser.
2. Navigate the calendar to pick a date, then add workouts via the form.
3. View stats on the dashboard; switch themes using the toggle in the header.
4. Export data for backups or sharing.
5. No setup or build tools needed—it's ready to run!
